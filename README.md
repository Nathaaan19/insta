# Instagram Clone

Um clone básico da interface do Instagram desenvolvido com Expo e React Native.

**Autor:** Nathan Rodrigues Pulchinelli

## Features

- Header com logo Instagram
- Post estático com foto de usuário (gerada por IA)
- Botões de interação (like, comment, share, bookmark)
- Suporte para tema claro e escuro
- Componentes reutilizáveis

## Como usar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o aplicativo:
```bash
npx expo start
```

3. Scan o QR code no seu celular com Expo Go ou escolha uma opção de emulador/simulador.

## Estrutura

- `app/_layout.tsx` - Layout principal
- `app/instagram-*.tsx` - Componentes do Instagram
